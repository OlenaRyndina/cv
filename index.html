Приложение состоит из 3 компонентов: селектор по месяцам, календарь, записи.
При загрузке приложения календарная сетка выстраивается согласно текущей дате. Компоненты общаются между собой через сервис даты и сервис данных. Данные списка дел хранятся в базе данных Firebase.
