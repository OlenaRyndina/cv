Академический проект:

Приложение Discovery country. Приложение для туристов и любителей изучать свой город. Информация по городским достопримечательностям: контактная, как добраться, отзывы, рейтинг популярности.

Используемые технологии:

  • frontend: Angular 13
  • backend: nest JS
  • База данных: mySQL (typeorm - библиотека для работы с БД)
  • XAMPP - интсрумент для работы с БД
  • Postman - для работы с запросами, которые имитируют запросы Клиента на Сервер
  • NGRX - контроль состояния приложения

В проекте реализованы следующие задачи:

  • lasy loading модулей
  • система аутентификации и авторизации пользователя
  • JWT-токен с обновлением каждые 15 мин
  • пароли шифруются с помощью bcrypt
  • при регистрации пользователя происходит асинхронная валидация (проверка логина на аутентичность)
  • данные пользователя при авторизации сохраняются в localStorage: при обновлении страниц или открытии приложения в новой вкладке данные о пользователе хранятся в течении 15 мин.
  • приложение отображает данные в зависимости от роли пользователя (авторизован или нет)
  • авторизованный пользователь может корректировать данные, добавлять новые и отправлять их на сервер, сохраняя их в БД. Неавторизованный пользователь получает данные без возможности их редактировать, только просмотр.
  • реализован фильтр данных по таблице
  • реализован компонент "Всплывающее сообщение", оповещающие пользователя о результате его действий: успех, ошибка или предупреждение.
  • реализован компонент Like/dislike: авторизованный пользователь может кликнуть на иконку с сердечком и добавить +1 к счетчику "нравится". При повторном клике происходит уменьшение счетчика "нравится" на -1. Данные уходят на сервер, автоматически обновляясь в шаблоне Клиента.
